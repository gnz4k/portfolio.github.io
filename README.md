# Portfolio

This is a portfolio website for me (Gonza).
Website: https://gnz4k.github.io/portfolio/

## Features

- **Home**: Displays the developer's name, skills, and experience.
- **Projects**: Showcases different projects with images and descriptions.
- **Skills**: Lists the developer's technical skills.
- **Contact**: Provides contact information and links to social media.

## Structure

The website uses a single HTML file (`index.html`) with embedded CSS and JavaScript.

### Key Components

- **Sidebar Navigation**: Allows users to navigate between different sections.
- **Search Container**: A search bar with an auto-complete dropdown.
- **Views**: Different sections of the portfolio (Home, Projects, Skills, Contact).
- **Project Cards**: Cards to display project details.
- **Setting Items**: Items to display contact information.

## Technologies Used

- **HTML**: For the structure of the website.
- **CSS**: For styling and layout.
- **JavaScript**: For interactive features like navigation, search, and animations.
